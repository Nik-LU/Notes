# Приложение заметок

Android-приложение для управления заметками с полным стеком современных технологий. Реализовано с акцентом на чистую архитектуру, реактивный UI.

## Запуск
1. Склонировать репозиторий
2. Открыть проект в Android Studio
3. Запустить конфигурацию `app`

---

## Стек
- **Язык:** Kotlin
- **UI:** Jetpack Compose
- **Архитектура:** Clean (data / domain / presentation), MVVM + StateFlow
- **Screens:** Single Activity + Navigation Compose
- **Внедрение зависимостей:** Hilt
- **База данных:** Room
- **Загрузка изображений:** Coil 
- **Асинхронность:** Kotlin Coroutines, Flow
- **Прочее:** Splash Screen API, Internal Storage

---

## Назначение проекта
Проект демонстрирует полный цикл разработки современного Android-приложения:
- **Clean Architecture** с чётким разделением слоёв.
- **MVVM** с ViewModel и реактивными **StateFlow**.
- Внедрение зависимостей с помощью **Hilt**.
- Реализовано на **Jetpack Compose** с навигацией через **Navigation Compose**.
- Локальное хранение данных с использованием **Room** и **корутин**.
- Загрузка изображений с помощью **Coil**.
- Реализован **Splash Screen API** и использование **Internal Storage** для хранения данных.
